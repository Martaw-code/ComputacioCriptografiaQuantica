**Exercici 3**: La incertesa no seria 2.5! Caldria fer el següent: </br>

**σ = √((Σ(x - μ)^2) / N)** </br>
on: </br>

μ = (1 + 2 + 3 + 4 + 5 + 6) / 6 = 3.5 </br>
Σ(x - μ)^2 = (1 - 3.5)^2 + (2 - 3.5)^2 + (3 - 3.5)^2 + (4 - 3.5)^2 + (5 - 3.5)^2 + (6 - 3.5)^2 = 17.5 </br>
σ = √(17.5 / 6) ≈ 1.707825 </br>
